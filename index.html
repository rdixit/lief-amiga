Symbol Color Key

Background tints follow the Fitzgerald Key — a standard AAC color system based on grammar role.

  • Yellow — Pronouns (I, you, he/she)
  • Green — Verbs / actions (eat, go, play)
  • Blue — Nouns / things (home, water, toy)
  • Orange — Location & prepositions (in, out, up)
  • Purple — Descriptors (happy, big, fast)
  • Red — Negation / repair (no, stop, can't)
  • Pink — Social phrases (that's funny)
  • Gold outline — Suggested next word

Left border color indicates the symbol's tab category (Core, Actions, Feelings, etc.).
